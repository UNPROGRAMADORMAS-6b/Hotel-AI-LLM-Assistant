🤖 Hotel AI-Assistant: Sistema de Reservas Inteligente
Este proyecto es un asistente virtual para WhatsApp que utiliza Inteligencia Artificial Generativa para automatizar la gestión de un hotel.
🛠️ Tecnologías Utilizadas

    Backend: Node.js 🚀

    IA: Google Gemini API (Modelo 1.5 Flash) 🧠

    Base de Datos: MySQL 8.0 🗄️

    Comunicación: WPPConnect (WhatsApp Web JS) 📱

    Entorno: JavaScript / Dotenv

🌟 Beneficios Principales

    Disponibilidad 24/7: Atención inmediata sin intervención humana.

    Escalabilidad: Capaz de manejar múltiples reservas simultáneas (Stress Tested).

    Fácil Integración: Conexión simple con sistemas de base de datos existentes.

🎓 Contribución Académica

Este repositorio está diseñado para servir de guía a estudiantes de la UCV y otras universidades que necesiten un ejemplo real de integración de IA con bases de datos relacionales para sus proyectos de fin de carrera.




### 💻 Instalación Rápida
1. Clonar el repo: `git clone https://github.com/UNPROGRAMADORMAS-6b/Hotel-AI-LLM-Assistant`
2. Instalar dependencias: `npm install`
3. Configurar `.env` con tus credenciales.
4. Ejecutar: `npm start`
